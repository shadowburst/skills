# Skills

This repository contains my personal collection of skills for AI-assisted development workflows.

These skills are based on Matt Pocock's skills repository, with modifications to better fit my own workflow, preferences, and project conventions. They should be treated as adapted working notes rather than a direct copy of the original source.

## Purpose

The skills in this repo are intended to help with recurring development tasks such as:

- test-driven development
- codebase architecture improvement
- specification writing
- code review
- diagnosis and debugging
- skill authoring and setup
